## 📁 How to download the Project 
<br/>

```bash 
    #clone the repository
     $ git clone https://github.com/VidalJunior05/jogo-da-velha-javascript.gitt
    
    # Enter directory
    $ cd jogo-da-velha-javascript

    # install dependencies
    $ yarn install

    # Initialize project
    $ yarn start
```
